# wanted-pre-onboarding-frontend pre-assignment

이 프로젝트는 원티드에서 주관하는 '원티드 프리온보딩 프론트엔드 인턴쉽'의 사전 과제 결과물입니다.

## 프로젝트 실행 방법

- 해당 git clone 후 프로젝트 디렉토리의 터미널에서 아래와 같이 실행할 수 있습니다.
```
npm install 
npm start
```
